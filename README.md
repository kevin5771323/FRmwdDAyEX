# 公寓管理系统 java506

## 项目概述

公寓管理系统是一款基于Spring Boot、MyBatis和Vue技术栈的软件，用于实现公寓管理的各项功能。本项目通过后端与前端的分离，为管理员和用户提供了一个易用、高效的管理与服务平台。

## 技术栈

### 后端技术

- Spring Boot
- MyBatis

### 前端技术

- Vue

## 运行环境

- 开发工具：IntelliJ IDEA / Eclipse
- 数据库：MySQL 5.7
- 运行方式：Spring Boot

## 功能模块

- 用户角色管理：后台可管理不同角色的账号权限，并允许自行创建账号。

## 系统角色

- 管理员：拥有最高权限，可管理所有用户和系统设置。

## 部署步骤

1. 配置好MySQL数据库环境。
2. 导入项目所需的数据库文件。
3. 使用IDEA或Eclipse运行本项目。
4. 打开浏览器访问 `http://localhost:9876/login` 进行登录。

## 核心亮点

- 完善的用户角色管理机制，保证系统安全与权限控制。
- 基于 Spring Boot，提供高效的后端服务。
- 使用 Vue 构建的前端界面，易于维护和扩展。
